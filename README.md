Movie App – Mobilna aplikacija razvijena u React Nativeu koja omogućuje korisnicima pregledavanje i pretraživanje filmova po naslovu, žanru i ocjeni. Aplikacija dohvaća podatke o filmovima putem vanjskog API-ja (The Movie Database API) te prikazuje detaljne informacije poput opisa, trajanja, ocjene i glumaca. Implementirano je dinamično pretraživanje, vizualno privlačno korisničko sučelje i responzivan dizajn prilagođen mobilnim uređajima. Aplikacija je izrađena korištenjem Expo alata i Tailwind CSS frameworka za brzu izradu modernog UI-a.
